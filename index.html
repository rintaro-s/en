This site is a site where you can see some of the thumbnails, titles, and summary columns of YouTube videos provided by Google, and depending on the environment, embedded YouTube videos may be displayed.

The rough explanation is displayed using html js css using the "yatsu" provided by Google that allows you to watch youtube videos.

For a detailed explanation (for developers), use the youtube data api v3, an api provided by Google that allows you to get youtube video information, get the api key from google cloud, and use javascript to get the youtube data api It is a mechanism such as specifying the v3 endpoint URL, enabling API requests using the api key, and viewing each video information
